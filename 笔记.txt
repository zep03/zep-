Node.js是基于 Chrome的V8 JavaScript 引擎构建的JavaScript运行环境。
Node.js不是新语言，也不是一个框架或者一个库，而是一个软件。
Node.js是一个 JavaScript 运行环境 ，说白了，就是用来运行js的。
Node 打破了过去 JavaScript 只能在浏览器中运行的局面。前后端编程环境统一，大大降低了前后端语言切换的代价。

为啥偏爱NodeJS？
让前端职责范围变大，统一开发体验。对前端开发人员来说，不需要再去学习额外的后端语言，就能够开发后端
在处理高并发、I/O密集场景性能优势明显


MongoDB是目前nosql中最热门的数据库，开源产品，基于c++开发。
是nosql数据库中功能最丰富，最像关系数据库的非关系型数据库
其优势在于查询功能比较强大，能存储海量数据，
缺点是比较消耗内存。


Mongoose 是在 node.js 异步环境下对 mongodb 进行便捷操作的对象模型工具。Mongoose 是 NodeJS 的驱动，不能作为其他语言的驱动。
Mongoose 有两个特点 ：
1、通过关系型数据库的思想来设计非关系型数据库
2、基于 mongodb 驱动，简化操作


Express 是一个基于 Node.js 平台，快速、开放、极简的 web 开发框架，它提供一系列强大的特性，帮助你创建各种 Web 和移动设备应用。
Express项目搭建：
1、新建项目文件夹
2、初始化项目 yarn init -y 或者 npm init -y
3、安装express，yarn add express 或者 npm install express
4、其次在项目文件夹中，创建一个名为 app.js 的文件
5、设置静态资源，开放public 和 node_modules目录
6、设置模板引擎（安装 art-template 和 express-art-template）
7、设置post请求方式的处理（安装body-parser）
8、添加cookie和session的配置（安装cookie-parser和cookie-session）
9、编写路由文件
10、把路由挂载到app对象中
11、监听端口
12、启动项目（node app.js）

实现的功能如下：
用户登录	用户输入电子邮箱、密码进行登录。如果不存在该用户，则跳转到注册页面。
用户注册	用户输入电子邮箱、昵称和密码进行注册。如果已经存在该用户，则提醒用户重新输入。
登录退出	用户可以退出当前登录的账号。
密码修改	用户在正确输入原始密码后，可对密码进行修改。
用户注销	用户可对当前账号进行注销。
个人信息修改	用户可对自己的昵称、性别、个性签名、出生日期等个人信息进行修改。
技术交流	查看最新帖子、热门推荐等数据
发布文章	用户登录后可发布文章
发表评论	用户登录后可对他人发表的文章进行评论
查看评论	用户点击某篇文章后可查看这篇文章相关的评论
